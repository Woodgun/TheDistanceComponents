    Should record the given analytic event, optionally starting a new session.

    Declaration

    Swift

    func sendAnalytic(event:AnalyticEvent, asNewSession session:AnalyticSession?)

    Parameters

    event

    The event to send.
    session

    If this value is not nil, this should begin a new analytics session with the given configuration, closing a previously opened session. This should be called whenever anyhting in the session configuration changes, e.g. user logs out / in.
  • sendAnalytic(_:) Default implementation

    Should record the given analytic event. A default implementation is given which calls sendAnalytic(_:asNewSession:) without starting a new session.
